WHAT WE OFFER?

A powerful Angular library that provides reactive, responsive, and beautiful charts for your applications. Built on top of Chart.js with full TypeScript support.

Get Started

Installation

npm install ngx-charts chart.js

or

yarn add ngx-charts chart.js

Quick Start

Import and use BaseChartDirective in your standalone component. Add it to your component's imports array.

imports: [BaseChartDirective]

Visit our GitHub repository for detailed documentation and complete examples.
